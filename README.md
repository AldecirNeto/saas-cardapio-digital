# SaaS Cardápio Digital 

Um sistema Multi-Tenant de Cardápio Digital desenvolvido para restaurantes. A plataforma inclui painel administrativo, Paineel da cozinha para gestão na cozinha,Painel do Caixa para controle de pagamentos, motor de cupons e captura de leads.

## Tecnologias Utilizadas

- **Backend:** Python (Flask)
- **Banco de Dados:** PostgreSQL (Hospedado no Supabase)
- **Frontend:** HTML5, CSS3, JavaScript 

## Principais Funcionalidades

- **Arquitetura Multi-Tenant:** Isolamento rigoroso de dados entre diferentes restaurantes.
- **Painel Admin:** Gestão completa de categorias, produtos e fluxo de pedidos.
- **Tela de Cozinha:** Acompanhamento de pedidos em tempo real.
- **Motor de Conversão:** Aplicação de cupons em 1-clique atrelado à captura de WhatsApp do cliente.
- **Segurança:** Proteção de rotas com decorators customizados e controle de sessão.
